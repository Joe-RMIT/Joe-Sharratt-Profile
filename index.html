Project Idea

Pomodoro Day Planner

Overview


The Pomodoro Day Planner is a productivity application that will be developed for Apple’s iOS. The application revolves around the ‘pomodoro’ productivity system. This system involves a set of timers which break tasks into short periods of focused and intensive work followed by a short break. Where the Pomodoro Day Planner will differ from similar iOS applications is its attempt to create an all-encompassing task-management system alongside the traditional Pomodoro timer. Features will include an integrated to-do list and a daily reflection journal.

Motivation


I have been experimenting with a variety of different productivity applications over the past few years and have yet to find the perfect tool for my workflow. As a result, I view the Pomodoro Day Planner as a solution to a personal problem.

As it stands, I use three different productivity tools every day. I use ‘Notion’ for my task management & to-do list, ‘Day One’ for my journaling and daily reflection, and ‘Emphasis’ as my Pomodoro timer. While these applications are good at handling their individual tasks, I would like to be able to create one application that I could work in for the majority of my workday.

Description


Pomodoro Day Planner is an iOS application that revolves around three core features. First and foremost, the application is centred around Francesco Cirillo’s ‘Pomodoro’ productivity system, which is a “simple yet effective tool for focused work with planned breaks in between” (Mandal, 2020). The system encourages the user to remain focused and ‘on-track’ when completing complex tasks by splitting a workday into multiple 25-minute sessions of focused work which are followed by 5-minute breaks. After every fourth completed 25-minute work session, the user takes a longer 15-minute break. These times will serve as the default settings for the timer in Pomodoro Day Planner, however the user will be able to modify the length of work and break sessions in order to tailor the application to their needs.

Second, the application will feature a simple to-do list. Considering that the application is centred around the idea of focused work, the feature-set of this to-do list will be kept deliberately stripped back compared with other to-do lists such as Apple’s ‘Reminders’ app. The to-do list in Pomodoro Day Planner will allow users to create a to-do list made up of tasks and sub-tasks that will be displayed with a strike-through when marked as completed.

Third, the application will feature a simple journaling tool that will prompt users each day at a set time with a variety of writing prompts designed to help users reflect on their successes of the day and strive to develop healthy working habits over time. These questions include:


  • Did you complete your to-do list today?
  • What was your biggest win today?
  • What long-term project did you work on today?
  • What did you have to reschedule?
  • What could you have done to be more productive?
  • What made your day better?
  • What is your most important task for tomorrow?
  • What did you listen to today?

Journal entries will be stored locally on a user’s iPhone and will be displayed within a built-in calendar.

Pomodoro Day Planner is designed to boost productivity and enhance the user’s ability to focus when completing complex tasks. With this in mind, the application will be designed with a minimalist aesthetic and will look to reduce the use of menus and button presses wherever possible. To achieve this, the application will feature three pages that will be accessed via swipe gestures rather than button presses. The main ‘pomodoro’ page will show a simple, circular countdown timer. Underneath this timer, the application will display the number of sessions that have been completed during the workday. The second page will feature the simple to-do list that will allow the user to create, modify and delete tasks. The Third page will display the journal and will allow users to create new entries or recall previous entries from a in-built calendar.

Tools and Technologies


Pomodoro Day Planner will be developed for iOS 15 and will be written in Apple’s ‘Xcode’ application using the ‘Swift’ programming language. As a result, the successful development of the application will require access to an Apple computer capable of running Xcode 13, as well as access to ‘Canva’ which is a free, browser-based graphic design tool. Finally, access to an iPhone with iOS 15 installed will be essential for testing the application prior to launch.

Skills Required


The development of this application primarily relies on a working knowledge of ‘Swift’, which is something I plan to develop throughout this course. However, it is important to note that while programming skills are an obvious necessity, they will need to be supplemented with additional skills to ensure the successful completion of the project. For example, the self-paced nature of software development requires a person with strong time management skills and a natural ability to maintain focus over prolonged periods of time. Further, software development is often inherently a collaborative effort. As a result, strong interpersonal skills are required to ensure that healthy, working relationships with teammates are created and maintained.

Outcome


In a broad sense, the application will be considered a success if it successfully solves the initial problems listed in the Product Overview. Namely, the Pomodoro Day Planner will have successfully filled the need for users seeking to boost their productivity through a single application that combines a traditional pomodoro timer, a to-do list, and a daily reflection journal. Further, successful development of the Pomodoro Day Planner will result in the developers registering for the Apple Developer Program and submitting the application for review in an attempt to get the app listed on the iOS App Store.
